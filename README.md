# Movie App
It's an Android Mobile Application to:
## Displays the a list of the best movies in 2024
Each movie entry display:
- Movie poster
- Movie name
- Rating
- Release date
- Add to favorites option

## Project functionality:
  * MVVM
  * Room
  * Retrofit
  * Koin
  * Lottie
  * RoundedImageView
  * Gson
  * Coroutine
  * Glide
  * Navigation component
  * minSdk 21
  * targetSdk 34
